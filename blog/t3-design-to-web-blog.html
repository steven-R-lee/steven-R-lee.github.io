what is a responsive site?

A Resposive Site is one that responds to the screen
size of the viewing device, a smartphone, a phablet, a laptop or a desktop
computer. On a technical note this metric is determined by the number of pixels
that the screen is made up of.

And why is responsiveness important?

Resposiveness is important because it impacts on users'
experiences, (UX). It adapts the websites features to ensure the users experiences
are more likely to be positive, i.e., being able to view all the essential features
on the page, catering for speedy downloads, etc.
If the user isn't impressed, they most likely won't return to your website.

what is mobile first design?

Mobile First is when the web designer considers small mobile
mobile devices first during the process of designing a website.

why is "mobile first" important?

Mobile First is important because it helps determine the baseline
experience for users of a website.
Small mobile devices aren't as powerful as desktop computers, generally, and the situation
is changing. Dealing with the technical requirements of small mobile devices first
ensures a minimization of frustration levels of such users, while allowing space enhancements
for users of more powerful devices.

what are frameworks?

Frameworks are packaged kits - processes and tools.
These allows web developers to rapidly implement websites.

what are the pros and cons of frameworks?

Pros:
  • allow for rapid website building, using templates
  • allow for consistency and easy for beginners
Cons:
  • use of templates limits flexibility
  • lack of scope for the web developer to attain unique style

what is a wireframe?

A Wireframe is a systematic method of laying out initial
webpages designs on a grid, and using boxification.
The grid is typically 12-by-12 in size.

why we use a wireframe?

It simplifies the process of translating design from ideas to screen.
