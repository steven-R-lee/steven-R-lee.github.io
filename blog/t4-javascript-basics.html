how does JavaScript compare to HTML and CSS?

HTML, CSS and JavaScript are the 3 fundamental programmiing languages used in constructing today's modern web-pages and web-apps.

  • HTML acts as placeholders for content on web-pages.
  • CSS provides the mechanism for styling the look and feel of web-pages.
  • JavaScript provides the means for bringing dynamism to web-pages.

HTML and CSS when used in combination can produce static web-pages, just like the pages you'd see in a book.

By adding JavaScript can produce interactivity between web-users (clients) and web-producers (servers). In doing so, static web-pages are transformed into interactive web-apps.

This interactivity comes at a price: JavaScript.
Transforming static web-pages into inteactive web-apps demands a more sophisticated and robust programming language than HTML and CSS. Utilisation od JavaScript allows web-developers to produce web-apps that support the delivery of both dellivering and receiving communicationsover the internet:a dialogue as opposed toa monologue.

explain control flows and loops using an example process from everyday life,
for example, 'waking up' or 'brushing you teeth'.

Control Flows are classically represented in JavaScript by if, if/else, if/else if/ else and switch: case/break/default commands. These are analogous to choice-making procedures in everyday life.

You are required to make decisions in your everyday life based on the options you have in front of you and on your desired outcome.
For Example: IF the bus is due to arrie within 15 minutes I wait for it and catch the bus home ELSE I will walk home.

Loops are classically represented in JavaScript by for, while, do while commands. These are analogous to repeating procedures in everyday life.

You are required to repeat actions in your everyday life based until a certain desired condition is satisfied.
For Example: If an exam consists of 100 questions each worth one mark and you need to attain a score of 70% to pass the exam then:
WHILE you have not answered 70 of the 100 exam questions correctly, keep attempting to answer questions correctly.

Explain the difference between accessing data from arrays and object literals

You might access the data within your data containers through two ways using JavaScript: From Arrays or thorugh Object Literals.

For example: objectName{"propertyName"} --- accessing through From Arrays means.
For example: objectName.propertyName --- accessing through Object Literals means.
These commands provide equivalent results.

explain what fucntions are, and why they are useful

In JavaScript, Functions allows you to group a series of commands together in order to perform a specific task. If different parts ofthe JavaScript code repeat the same task over and over again, you can re-use the function (rather than repeating the same set of instructions).

For example: In everyday life you might brush your teeth after everytime you eat. Such a function required you to commit a series of actions: go to bathroom, find tooth brush and find toothpaste, open tooth paste tube, apply toothpaste to toothbrush, brush teeth, etc. You commit that series of actions to your memory in the form of a stored function. And instead of listing out all the actions required for cleaning your teeth each time you might just recall the function: 'CLEAN TEETH'.
