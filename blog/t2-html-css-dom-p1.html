what is HTML, CSS and DOM?

There are essential three components that are needed for building a webpage and these are:

  • HTML
  • CSS
  • DOM

What is HTML?

HTML is formally known as Hyper Text Markup Language. It is the programming language of the Web. It consists of a number of elements and tags which act as placeholders that allow the web developer to place the content on a webpage.

what is CSS?

CSS is formally known as Cascading Style sheets. It an accompanying programming language for the Web. It provides the style, the look, of a webpage.

what is DOM?

DOM is formally known as Data Object Model. It describes the model by which the elements and tags HTML are set out on the webpage - typically as a tree.

what is meant by boxifying design?

Boxifying Design is when the web developer will take the initial design for a webpage and begin to turn it into code. This is achieved by drawing boxes around each element on the design page. This task allows the web developer to start identifying the elements, or placeholders, within the HTML code.

what is the box model?

The Box Model is thus, when the web developer transforms a page design into a formal set of boxes prior to coding.
