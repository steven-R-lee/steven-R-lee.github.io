parentheses ()

Parentheses are written as () in JavaScript.
They have three major uses in JavaScript with:

  1. Function Calls
  2. Conditional Statements
  3. Order of Operations

In JavaScript, Function Calls are used to call a function which is a block of computer code again and again from inside a programme. The function with its accompanying () perenthese will be assigned a name by the coder who will then be able to call it from anywhere with the programme. The () parentheses may sometimes be empty but often they'll contain arguments - values that will be feed into the function.

In JavaScript, Conditional Statements are used to control the flow of a programme. These consists of "if", "elseif" and "else" statements. The condition componet of these commands will be contained within the () parentheses.

In JavaScript, Order of Operations that's mathematically operations can be overridden by the applications of () parentheses. i.e. ((1+2)*3) is not equivalent to (1+2*3).

bracket []

Brackets are written as [] in JavaScript.
The primary use of [] brackets in JavaScript is for access data objects. Data objects can be thought of as containers with data inside of them. These are typically in the form of lists or arrays in the common parlence.

Given an array of var myArray = [1,3,5,7,9], the forth number in the list can be accessed through myArray[3], remembering the first number, (or element) in the array has an index of [0].

braces{}

Braces are written as {} in JavaScript.
The two principle uses of {} braces in JavaScript are for containing object literals at the point of their declaration and to act as benign placeholders for blocks of code to make the code more readable.

single quotes '' and double quotes ""

There are no hard and set rules around the use of single quotes '' and double quotes "" in JavaScript.
THowever, it is important to be your consistent with them when coding in JavaScript, just as you would be when writing in English.
